# Railway settings fix

В этой сборке Railway Variables больше не управляют рабочими настройками бота.

В Railway оставь только:

- `BOT_TOKEN` — токен Telegram-бота от BotFather
- `ADMIN_IDS` — Telegram ID админа, можно через запятую

Все остальные настройки меняются внутри Telegram через `/settings`:

- авто-сигналы ON/OFF
- порог проходимости
- таймфрейм
- интервал скана
- cooldown
- максимум сигналов за скан
- биржа
- фильтр тренда
- умный алгоритм
- нейро-оптимизатор
- супер-сделка
- только BTC/ETH
- наклонки
- улучшения торговли
- автоторговля, маржа и TP-закрытие

Настройки сохраняются в `data/settings.json`. На Railway для постоянного хранения между redeploy/restart лучше подключить Volume к `/app/data`.

API-ключи для LIVE-торговли теперь добавляются через Telegram командой `/api_set`, а не через Railway Variables.
