# Manual Scan Percent Analysis Fix v1022

Исправление ручного ввода монеты:

- фото графика теперь получает подпись с полным ручным анализом;
- в подписи и отдельном тексте показываются:
  - проходимость сигнала в процентах;
  - вероятность LONG в процентах;
  - вероятность SHORT в процентах;
  - текущий перевес LONG/SHORT;
  - наклон линии в процентах за окно свечей;
  - наклон за свечу в процентах;
  - Entry, SL, TP1 в процентах от входа;
  - RR до TP1;
  - вывод: выше/ниже порога авто-сигнала;
- если найден строгий наклонный уровень, его вероятность и наклон не затирают ручные проценты;
- ручной анализ не запускает автоторговлю и не меняет авто-скан;
- версия бота обновлена до 1022.
