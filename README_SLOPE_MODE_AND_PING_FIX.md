# Исправление режимов наклонок и кнопки Ping

Что изменено:

1. В настройках `/naklonki` и `/settings -> Наклонки` теперь есть три режима:
   - `OFF` — наклонки выключены, сигналы идут по обычной логике.
   - `Только наклонки` — сигнал/авто-сделка отправляется только если найден сильный наклонный уровень. Общий порог проходимости не обязателен.
   - `Порог + наклонки` — строгий режим: базовая проходимость должна быть не ниже текущего порога, и наклонный уровень тоже должен пройти свои условия. Оба фильтра обязательны для сигнала и открытия авто-сделки.

2. Режим сохраняется в `/app/data/settings.json` как `SLOPE_LEVELS_MODE`.

3. Добавлена команда `/ping` и кнопка `🏓 Ping` на клавиатуре. Она показывает, что бот жив, скорость ответа и аптайм процесса.

Для Railway оставляй активными только:
- `BOT_TOKEN`
- `ADMIN_IDS`

Остальные настройки меняй внутри Telegram.
