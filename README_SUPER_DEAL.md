# 🔴 Супер сделка

Добавлен отдельный режим строгих сигналов/сделок.

## Как включить

Через Telegram:

```text
/settings → 🔴 Супер сделка → ON
```

Или командой:

```text
/super_deal
```

Или через Railway Variables / .env:

```env
SUPER_DEAL_ENABLED=true
SUPER_DEAL_MIN_PROBABILITY=97
SUPER_DEAL_RAW_PROBABILITY_MIN=95
SUPER_DEAL_TREND_SCORE_ABS=7
```

## Что делает режим

Когда `SUPER_DEAL_ENABLED=true`, бот отсекает обычные сигналы и обычные авто-сделки. Сигнал проходит только если выполнены максимальные условия:

- базовая проходимость до супер-фильтра не ниже `SUPER_DEAL_RAW_PROBABILITY_MIN`, по умолчанию 95%;
- для LONG тренд должен быть бычий и `trend score +7`;
- для SHORT тренд должен быть медвежий и `trend score -7`;
- итоговая проходимость супер-сделки выставляется в диапазоне 97-99%.

## Как выглядит сообщение

Супер-сигнал начинается так:

```text
🔴🔴🔴 Внимание, есть супер сделка!
🔴 Супер-сигнал: ENA_USDT / LONG
Проходимость: 98%
```

В Telegram Bot API нельзя реально покрасить текст сообщения в красный цвет, поэтому бот выделяет такие сигналы красными эмодзи 🔴 и словом «Внимание».

## Важно

Супер-сделка — это самый строгий технический фильтр, но не гарантия прибыли. В LIVE-режиме всё равно используй минимальный объём, защитные ордера, синхронизацию и риск-менеджмент.
