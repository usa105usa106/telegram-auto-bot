Telegram Signal Bot — MEXC/BingX + авто-торговля

Что добавлено:
- /settings -> настройки прямо в боте
- Биржа: MEXC Futures / BingX Futures
- Таймфрейм, проходимость, интервал скана
- Автоторговля: OFF / PAPER / LIVE
- Маржа/объём сделки в USDT
- Авто-закрытие всей сделки по SL или выбранному TP1/TP2/TP3
- /api, /api_set, /api_clear для ключей
- /trades для активных авто-сделок

Как добавить API ключи через Telegram:
/api_set MEXC API_KEY API_SECRET
/api_set BINGX API_KEY API_SECRET

На бирже включай только Read + Trade. Withdraw/вывод средств НЕ включать.

Безопасный порядок запуска:
1) В /settings включи PAPER и проверь несколько сигналов.
2) Поставь маленькую сумму в /margin, например /margin 5.
3) Только после тестов добавь API ключи и включи LIVE.

Важно:
- LIVE использует рыночные ордера через CCXT.
- Авто-закрытие сделок работает ботом через мониторинг цены. Это не гарантирует биржевой stop-loss, если бот/Railway упадёт.
- Используй минимальные суммы и отдельный API-ключ без вывода средств.
